# 排序算法总结
|          | 时间复杂度       | 是否稳定排序 | 是否原地排序 |
| :------: | :--------------: | :----------: | :---------:  |
| 冒泡排序 | o(n<sup>2</sup>) | 是           | 是            |
| 插入排序 | o(n<sup>2</sup>) | 是           | 是            |
| 选择排序 | o(n<sup>2</sup>) | 是           | 是            |
| 快速排序 | o(nlogn)         | 否           | 是            |
| 归并排序 | o(nlogn)         | 是           | 否            |
| 桶排序   | o(n + k), k 为数据范围 | 是     | 否            |
| 计数排序 | o(n)             | 是           | 否            |
| 基数排序 | o(d * n), d 为维度 | 是        | 否            |
